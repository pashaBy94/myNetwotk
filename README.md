## Описание
<p>Социальная сеть реализована на React/Redux/TypeScript</p>
<p> Реализованы особенности:
- Перенаправление на форму логинизации не зарегистрированных пользователей;
- Страница заглушка при возникновении ошибок сети;
- Загрузка фото в профиль с компьютера;
- Редактирование контактов о пользователе;
- Пагинация;
- Фильтр друзей;
- Добавление пользователей в друзья;
- Captcha google, captcha картинка;
- Валидация формы;
- Аудио плеер;
- Мини игра.
</p>
